# Загрузчик статей WeChat

Этот набор инструментов является упрощённой версией, основанной на проекте [qiye45/wechatDownload](https://github.com/qiye45/wechatDownload), разработанной для помощи пациентам и их семьям в быстрой загрузке и сохранении статей из публичных аккаунтов WeChat, связанных с информацией о заболеваниях поджелудочной железы.

**Особая благодарность оригинальному автору [qiye45/wechatDownload](https://github.com/qiye45/wechatDownload) ❤️ за оригинальное творение. Пожалуйста, поддерживайте/форкируйте/звёздите оригинальный проект!**

## Обзор архитектуры

### Упрощённый дизайн

- Удалена зависимость от локального сервиса MCP - этот набор инструментов можно развёртывать напрямую на Lobster/Hermes.
  
**Примечание для окружения Linux:** Поскольку Linux не может запускать клиент рабочего стола WeChat для Windows/macOS, действуют следующие ограничения:
- ❌ Локальный MCP (`http://127.0.0.1:4545/mcp`) **НЕ доступен**
- ✅ **Всегда используйте удалённый MCP**: `https://changfengbox.top/api/mcp`

### Поддерживаемые функции

| Функция | Имя инструмента | Статус |
|---------|------------------|--------|
| Загрузить одну статью | `wechat` | ✅ Доступно |
| Загрузить коллекцию/альбом | `wechat_collection` | ✅ Доступно |

### Устаревшие функции (не адаптированы к общим потребностям пользователей)

| Функция | Имя инструмента | Статус |
|---------|------------------|--------|
| Массовая загрузка всех статей с публичного аккаунта | `batch_download_articles` | ❌ Требует локальной настройки |
| Получить ключ публичного аккаунта | `get_public_account_id` | ❌ Требует локальной настройки |
| Экспортировать метаданные статей в CSV | `export_article_data` | ❌ Требует локальной настройки |
